# CS238PeerReviews
Repo that contains all the files for handling the peer reviews for the CS238 final project. Before you run the code, download the zip file containing all the reports and peer reviews from Google Drive (https://drive.google.com/file/d/1jyrubGxm1CpeF1_Aaqoa2PaVOWkmK7V1/view?usp=sharing) and unpack them in the main repository folder. You should see five additional folders in there now:
- project_files
- peer_review_1_files
- peer_review_2_files
- processed_peer_reviews
- publishable_reports

All the code for the peer review assignment and also for returning the peer reviews to the project authors can be found in 'parse_data.py'.
